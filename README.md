# Cuti Home Catania

Sito vetrina statico e responsive per Cuti Home.

## Pubblicazione con GitHub Pages

1. Carica tutti i file di questa cartella nella root della repository.
2. Apri **Settings → Pages**.
3. In **Build and deployment**, scegli **Deploy from a branch**.
4. Seleziona `main` e la cartella `/ (root)`.
5. Premi **Save**.

Dopo alcuni minuti il sito sarà disponibile su:

`https://devious810.github.io/Cuti-Home-Catania/`

## Struttura

- `index.html`
- `style.css`
- `script.js`
- `assets/images/`

Le immagini sono già ottimizzate in WebP.
